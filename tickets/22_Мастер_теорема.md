# Мастер-теорема

## 1. Определение

Мастер-теорема предоставляет асимптотическое решение для рекуррентных соотношений вида:

```math
T(n) = aT\left(\frac{n}{b}\right) + f(n)
```


```math
T(n)=aT(nb)+f(n),a≥1, b>1T(n) = aT\left(\frac{n}{b}\right) + f(n), \quad a \ge 1, \ b > 1 T(n)=aT(bn​)+f(n),a≥1, b>1
```
