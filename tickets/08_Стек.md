## 8. Стек  

### Определение

**Стек (Stack)** — это абстрактная структура данных, работающая по принципу  
**LIFO (Last In — First Out)** — _последний вошёл, первый вышел._

> Пример: стопка тарелок — снимаем сверху, добавляем сверху.

### Основные операции
1
| Операция           | Описание                              | Время    | Память   |
| ------------------ | ------------------------------------- | -------- | -------- |
| `push(x)`          | добавить элемент `x` на вершину стека | **O(1)** | **O(1)** |
| `pop()`            | удалить верхний элемент               | **O(1)** | **O(1)** |
| `top()` / `peek()` | вернуть верхний элемент (не удаляя)   | **O(1)** | **O(1)** |
| `empty()`          | проверить, пуст ли стек               | **O(1)** | **O(1)** |
| `size()`           | вернуть количество элементов          | **O(1)** | **O(1)** |
| `find(x)`          | найти элемент x в стеке               | **O(n)** | **O(1)** |


### Использование стека

| Применение                   | Пример                             |
| ---------------------------- | ---------------------------------- |
| Хранение истории действий    | отмена действий в редакторах       |
| Проверка правильности скобок | `((a+b)*c)`                        |
| Обход в глубину (DFS)        | стек заменяет рекурсию             |
| Перевод выражений            | из инфиксной в постфиксную форму   |
| Вычисление выражений         | с помощью обратной польской записи |
| Обратный порядок элементов   | например, реверс строки            |
| Подсчет | итоговый ключ длины k, полученный после удаления n - k цифр. |
